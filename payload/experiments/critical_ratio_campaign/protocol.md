# Protocol

1. Sweep:
   - governance_capacity
   - artifact_pressure
   - constraints
   - trust_continuity

2. Compute:
   U = (capacity * continuity) / (pressure * constraints)

3. Measure:
   viability_margin = (capacity * continuity) - (pressure * constraints)

4. Estimate:
   - collapse boundary
   - critical threshold Uc
   - baseline separation quality
