StegVerse Observatory Upgrade

Adds real-time experiment monitoring and replay capability.

Components:
- state_stream.py
- timeline_recorder.py
- replay_engine.py
- phase_animation.py
- live_dashboard.py

Usage:

Watch experiment live:
python observatory/live_dashboard.py

Record timeline:
python observatory/timeline_recorder.py

Replay:
python observatory/replay_engine.py

Animate phase map:
python observatory/phase_animation.py
