# Claims

## Proven by demo
- The critical ratio campaign computes the candidate invariant `U` as specified.
- The experiment protocol can be rerun with explicit configuration and expected outputs.
- The validation scripts can check deterministic and statistical output artifacts.

## Supported by demo
- Within this simulation family, `U = (capacity * continuity) / (pressure * constraints)` can be compared against simpler baselines.
- Boundary detection and threshold fitting can be repeated and summarized.

## Not yet proven
- Real-world universality across domains
- External validity beyond the sandbox model family
- Full physical-law generalization
