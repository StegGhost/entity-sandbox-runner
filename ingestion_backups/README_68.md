# StegVerse Sandbox

StegVerse Sandbox is the research and prototyping environment for the broader StegVerse / StegGhost ecosystem.

It provides a governance-first, plane-based runtime for:
- governed experiment execution
- phase-space and invariant discovery
- evidence and receipt logging
- deterministic ingestion of upgrade bundles
- sandbox-as-a-service extensions

## Primary roles

- host the core research engine
- document the runtime and architecture
- provide machine-readable structure for automation
- support reproducible discovery campaigns

## Key files

- `system_manifest.json` — machine-readable architecture map
- `docs/architecture/PROPOSED_BUILD.md` — human-readable architecture specification
- `docs/architecture/ARCHITECTURE_DIAGRAM.md` — quick reviewer-facing diagram
- `docs/research/RUNTIME_THEORY.md` — GCAT / runtime theory notes

## Recommended companion repositories

- `StegVerse-org/StegVerse-SDK` — SDK / client interface
- `StegVerse-org/StegVerse-demo-suite` — experiment protocols, demo campaigns, reviewer-facing reproductions
