# governed_agent_passport_v1

This is an ingestion-ready ZIP bundle for adding governed agent passport support.

## Install intent

Drop the bundle into your ingestion pipeline and map the included folders into the target repo root.

## Included directories

- `install/`
- `config/`
- `docs/`

## Manifest

The bundle manifest includes both `version` and `bundle_version` for compatibility with stricter verifiers.
